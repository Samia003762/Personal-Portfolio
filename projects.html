Cinema Booking System

This is a full-stack web application built using PHP, MySQL, JavaScript, HTML and CSS. It includes both a customer and admin side, with features such as:

  • User registration and login with hashed passwords
  • Movie listings and dynamically loaded showtimes
  • Interactive seat selection using JavaScript and PHP
  • Full relational database: users, movies, showtimes, seats, bookings, booking_seats
  • Admin dashboard to add movies, manage showtimes and edit seat layouts

This project taught me how to restructure a system after recognising limitations. I started with a Python Tkinter idea, then rebuilt the entire project in PHP after realising the advantages of web-based systems. I also strengthened my debugging skills when fixing SQL syntax issues and designed the interactive seat map independently.

Movie Data Analysis Capstone Project

As part of my Google Data Analytics Certificate, I completed a capstone project analysing a large movie dataset to explore relationships between ratings, genres and profitability. I used SQL (MariaDB) to clean and transform raw data, resolve data import issues, and create new analytical metrics such as profit calculations.

One of the biggest challenges was debugging a CSV import error where half the dataset was missing. After investigating discrepancies between datasets, I successfully re-imported and validated over 5,000 records. I also transformed complex genre data stored as structured text into a usable column, allowing meaningful grouping and analysis.

Using SQL queries, I analysed trends such as profit by genre, ratings versus profitability, and long-term industry trends. I then exported results into Google Sheets to create visualisations including bar charts, scatter plots, and trend analysis graphs designed for non-technical audiences.

This project strengthened my data cleaning, debugging, analytical thinking, and data storytelling skills, while giving me experience completing the full data analysis lifecycle from raw dataset to insight presentation.
